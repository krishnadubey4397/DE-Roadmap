Project: Public API Data Aggregator.
Description: Fetch data from a free public API for posts and comments. e.g. Twitter/Facebook
Tasks:
1. Write a Python script to fetch all posts and all comments.
2. For each post, count the number of comments associated with it.
3. Use Pandas to create a DataFrame of posts, including a new column for the comment count.
4. Save the final enriched DataFrame to a CSV file named posts_with_comment_counts.csv.
